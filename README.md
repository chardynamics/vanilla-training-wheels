<h3 align="center">The New Order: Vanilla Training Wheels</h3>

  <p align="center">
    As the name suggests, this is not a complete port of HOI4's vanilla GUI system, but rather a "training wheels" variant, where certain aspects have been ported from HOI4 Vanilla. The pictures provided show where the most change has occured, but bits of change are around the entire mod. The reasoning for this is that there are just too much aspects of the game that have this wireframe style (GUIs, economy, etc.) that would be too hard for a single person to change. Thus, I decided on this compromise, plus I think the original GUI is good and eventually this mod will get obsolete so this serves as "training wheels" for the true wireframe style.

Some things to note, I created this without any DLCs and I haven't tested it out after starting the game as I just haven't felt like doing a TNO playthrough in a while, but there is absolutely nothing that should happen to break it. If there is, I have a discord server [discord.gg] for a mod I've been working on with a North America map centered around a US collapse if the Soviets and the US switched sides called Land of the Divided that you could join and tell me. Thanks!

Credit to the original TNO vanilla mod by CPR which I based this off of, and of course the TNO mod.
